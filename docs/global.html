(constant) HeuristicsData

Data of the heuristics.

Used in user story uploading page to draw and show the available heuristics and also in the heuristic switching modal of the playground page.

New heuristics can easily be implemented in the frontend side through this array. Simply add a new object to the HeuristicsData array.

Properties:
Name Type Description
alt string

alternative text to display if the image in not available
value string

value of the heuristic ("h1", "h2", "h3"). Each value should be unique
text string

text to define the heuristic. Shown below the image in user story loading page and also in the heuristic switching button in playground
src *

image source
imageShape height:number

shape to be used in the user story uploading page This data is used for creating heuristic choice elements ReactUSUpload
