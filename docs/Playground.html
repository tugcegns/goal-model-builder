componentWillReceiveProps(newProps)

Does the following:

  • If the jsonExportClicked is true, then converts the paper to JSON and downloads it. Sets the jsonExportClicked back to false.
  • If uploadedObject is not undefined, it means that a new goal model has just been imported in one of the two ways: It is either imported as JSON or user story set. If uploadedObject has .cells attribute, then it is a JSON import and is treated as such. Otherwise it is a user story set processed in the backend and is used to create a graph using the createGraph method.

processLabel(label, elementType)

Process the label before drawing the element.

If type of the element is role, then it replaces every white space character with a new line character.

If the type of the element is goal:

  • If the label is too short (example: 'Goal'), corresponding goal element is too small. Adds whitespace to pad the string and draw a larget goal element
  • If the label has too many words, then the string becomes too long. Replaces every two white space character with a new line character
